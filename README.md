### Task : Keystroke Monitor (Keylogger)
A Python-based keystroke logging application developed during my internship at Arch Technologies using the `pynput` library.

**Features:**
- Monitors local keyboard events in the background.
- Logs characters and symbols to a local text file (`key_log.txt`).
- Handles special system keys (e.g., Space, Enter, Ctrl) gracefully using exception handling to prevent crashes.
- Implemented for authorized endpoint monitoring and security auditing purposes.
